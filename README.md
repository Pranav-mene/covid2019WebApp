## Installation

1.clone the Repository

```git clone https://github.com/Pranav-mene/covid2019WebApp.git

2.change directory

```cd covid2019WebApp

3.Install node package manager(NPM)

```npm install

4.Start the Application

```npm start or npm run dev (According to package.json scripts)
